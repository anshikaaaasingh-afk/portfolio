# 💼 Personal Portfolio Website (anshika singh)

A clean, professional, and fully interactive personal portfolio website built with **HTML, CSS, and JavaScript** — no frameworks needed.

![Portfolio Preview](preview.png)

## ✨ Features

- 🔤 Typing animation in the hero section
- 🌙 Dark / Light mode toggle
- 📍 Active nav highlight on scroll
- ✨ Hover effects on skills, cards & contact links
- 🎞️ Fade-in section animations
- 🟢 Pulsing "Open to opportunities" status
- 📱 Fully responsive (mobile-friendly)
- 🖼️ Professional photo embedded

## 🗂️ Project Structure

```
portfolio/

```

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

### 2. Open locally
Just open `index.html` in your browser — no server needed!
```bash
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### 3. Customize it
Edit `index.html` and replace:
- `Your Name` → your real name
- `your@email.com` → your email
- GitHub & LinkedIn links → your profiles
- University & school names
- Project titles and descriptions

## 🌐 Deploy on GitHub Pages (Free Hosting)

1. Push code to GitHub (see below)
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select `main` branch → **Save**
4. Your site is live at: `https://yourusername.github.io/portfolio`
